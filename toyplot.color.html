Defines the set of colors used by the map.

Values in the map domain are mapped to the colors in the underlying palette. Values outside the domain are clamped to the minimum / maximum domain values.

Notes

Linear maps are displayed as a color preview of their domain when viewed in an IPython notebook.

class toyplot.color.Palette(colors=None, reverse=False)[source]¶

Bases: object

Storage for a collection of related color values.

Parameters:

colors : sequence of color values, optional

Specifies the list of color values to store in the palette. The colors may be an Nx4 matrix of RGBA values, an Nx3 matrix of RGB values, or a sequence containing any combination of string CSS color values, RGBA tuples, or RGB tuples. If colors is unspecified, the palette will be initialized with a default collection of colors.

reverse : boolean, optional

If True, the values in colors will be stored in reverse order.

Notes

You can iterate over the colors in a Palette using normal Python iteration.

Palettes are displayed as a collection of color swatches when viewed in an IPython notebook.
